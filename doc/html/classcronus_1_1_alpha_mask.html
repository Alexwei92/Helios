Detailed Description

AlphaMask geometric object.

An AlphaMask is a Patch that is masked to create a generic polygon shape using the alpha channel from a PNG image file. Its position is given with respect to its center. AlphaMasks can only be added through the Context member function Context::addAlphaMask(). To retrieve a AlphaMask from the Context, see Context::getAlphaMask().
