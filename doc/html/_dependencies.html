Installing dependent libraries

Simple command-line tools are available to easily install dependent software libraries. The appropriate commands will be given in the documentaion to install any required libraries.

For Mac OSX, homebrew is a great package installer. Homebrew itself can be installed via a single command:

 $ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

For Unix systems, the apt-get command can be used, which comes pre-installed.

Helios dependencies

CMake Package

Components of Helios are compiled using the CMake package (www.cmake.org). Version 2.4+ is required. CMake can be also installed via the Unix command line "$ sudo apt-get install cmake".

C++ Compiler

You will also need a C/C++ compiler installed. Most systems come with a C compiler pre-installed, but some may need a C++ compiler. This can easily be accomplished via the command line "$ sudo apt-get install gcc g++".

Package
CMake
$ brew install cmake
$ sudo apt-get install cmake
See www.cmake.org for the graphical version of CMake.
GNU C/C++ Compiler
$ brew install gcc
$ sudo apt-get install gcc g++
Unknown

Note that individual plugins (see Writing Plugins) may have their own library dependencies. Consult their documentation for dependency information.
